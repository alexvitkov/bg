# бг - БЪЛГАРСКИЯТ ЕЗИК ЗА ПРОГРАМИРАНЕ

## Примерен код
факториел.бг:
```
функцийка факториелче(н):
    в случай че н <= 1:
        повърни 1
    в противен случай:
        повърни н * факториелче(н - 1)

изпринтирай(факториелче(100))
```
